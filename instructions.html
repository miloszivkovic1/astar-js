Instructions

This is a JavaScript implementation of A* algorithm. All of the options on the side panel are explained as follows:

New World Completely resets all nodes and creates a new world. Redraw World Redraws the world and re-animates the character. Show Borders Shows black borders around each node for a better view. Distance Function Changes the heuristics function the algorithm is going to use next time it is initiated. The available functions are:

  1. Euclidean d=(x1-x2)2+(y1-y2)2
  2. Euclidean Free
  3. Manhattan d=|x1-x2|+|y1-y2|
  4. Chebyshev d=max(|x1-x2|,|y1-y2|)
  5. Chebyshev Free
Free functions allow "squizeeng through" obstacles.
